# 二、准备工作

没什么好说的，一个可以正常使用的U盘，没必要太好，随便买个usb2.0的都可以，当然3.0的话安装会快一点哦。
## 01、制作PE启动盘
先安装WePe，这是一个纯净的PE系统，不会给你安装任何全家桶服务，虽然很就没有更新了，但是功能完善，依然可用。
[WePE Downlload](http://www.wepe.com.cn/download.html)
下载第一个微PE工具箱V2.0，附带的win10系统在下面也有，一并下载即可。我是机械革命z2自带的出厂系统镜像，好处是不用再去想办法激活，激活也好办啦，KMSpico了解一下。哦，要低调，存在我深深的百度网盘里。

## 02、用到的链接
机械革命深海幽灵z2原装出厂系统 链接: https://pan.baidu.com/s/1WuxrLRpeydZhmKt6wz8gxw 提取码: KMSp
KMSpico  V10.2.0 链接: https://pan.baidu.com/s/1B_UijZxMzkeiRnMgAgGNrA 提取码: KMSp

*其实吧，机械革命原装系统就只是特化了一下oem，也就是电脑，属性那里显示的图标，田牌被换成了机械革命logo，其他其实没有任何优化。而且是家庭版。没有特殊需求的下载wepe那个网页下面的镜像文件就好。*

## 03、安装系统
后面都是直接傻瓜式安装，wepe会格式化你的U盘，别作死放什么重要文件在U盘里头。

### i. 设置启动项
PE盘制作好把机械革命的系统镜像文件放进去。然后重启，按F2，进入bios界面，在BOOT选项里把#option1设置为USB启动项，应该是你的U盘的名字，2.0,3.0这些字样的那个。F4保存退出重启就进入PE系统了。
### ii. 装载iso文件
PE里找到镜像iso文件，右键装载，就进入安装界面了，机械革命的iso就是这么简单，后面选择自己要安装的硬盘，格式化以后安装比较好，否则原系统残留什么的很难受。
### iii.注意事项
**话说在前面，因为ubuntu的引导是安装在windows系统盘内的，要是格式了系统盘那么ubuntu的引导也就挂了，到时候最快的办法还是重装，没事，我已经先写好了ubuntu的重装日记。**



# 三、 安装开发环境
## 01：安装配置java
[Java8下载](https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
我这里选择的是**jdk-8u201-windows-x64.exe**
下载完成后双击安装，这里先说一下楠皮教我的理念，我之前的意识一直停留在机械硬盘时代，就想着把软件装在非系统盘，但现在电脑一般都把系统装在固态硬盘，所以我完全没必要去忧心开机速度。最合理的方式就是把那些可以安装的软件都放在系统盘，而自己写的工程文件和其他文档电影之类的放在非系统盘，这样一来就算系统崩了，下次大不了格式化了系统盘，重装一下，工程文件都还在，开发环境配好就和原来一样了。

双击安装，路径默认。后面全部傻瓜式安装。
安装完成后win+R，输入cmd打开命令行窗口，输入java -version查看java安装版本。

环境变量配置
1.我的电脑→右键→属性→高级系统设置→环境变量（可以右键固定在任务栏，以后会经常用到的）

```JavaScript
JAVA_HOME  
C:\Program Files\Java\jdk1.8.0_201
```

新建一个变量名为“JAVA_HOME”的系统变量，变量值为“C:\Program Files\Java\jdk1.8.0_201”（jdk的安装目录，根据个人所安装的目录修改）



![JAVA_HOME](https://s2.ax1x.com/2019/04/08/A4fnV1.png)


2.再新建一个变量名为“CLASSPATH”，变量值为“.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;”的系统变量，注意前面的点号和分号都是有的


```JavaScript
CLASSPATH
.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
```

![CLASSPATH](https://s2.ax1x.com/2019/04/08/A4fuUx.png)
3.打开“Path”系统变量，点击新建，添加“%JAVA_HOME%\bin”和“%JAVA_HOME%\jre\bin”两个系统变量。Path使得系统可以在任何路径下识别java命令


```JavaScript
%JAVA_HOME%\bin
%JAVA_HOME%\jre\bin
```

![Path](https://s2.ax1x.com/2019/04/08/A4febR.png)

04验证：点击保存后，按window+R或者点击运行输入cmd，打开命令行窗口,分别输入java和javac
```JavaScript
java -version
java
javac
```
都正常运行即代表java已经正确安装(这一步都出错的不要往下看了，java都安装不来的不配看我的重装日记）

![java](https://s2.ax1x.com/2019/04/08/A5neaD.png)
![javac](https://s2.ax1x.com/2019/04/08/A5nZVO.png)

# 五、遇到的bug归纳（不定时更新）
## 01 复制粘贴时进度条窗口不显示
打开cmd或者powershell
输入命令：
```
regsvr32 shell32.dll
```
回车即可。
## 02 玩游戏时显示.dll文件丢失

下载一个directX修复工具增强版修复即可
[directX修复工具增强版 v3.8](https://www.crsky.com/soft/31471.html#down)
